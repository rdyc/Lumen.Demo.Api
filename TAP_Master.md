#TAP RESTfull API

## Run database migration

`$ php artisan migrate --path=/database/migrations/TAP --database=<database_connection_name>`

The variable of "database_connection_name" can be found in:

`~/config/database.php`

## Run unit test
`$ vendor/bin/phpunit`
